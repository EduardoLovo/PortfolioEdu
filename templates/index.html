Projetos em Python

Projeto Detetive

Esse projeto totalmente em Python é um programa totalmente em Python que faz perguntas ao usuario sobre um crime e conforme as respostas, diz se o usuario que esta repondendo é "inocente", "suspeito", "cumplice" ou "culpado.

Projeto Jo-Ken-Po

Um jogo de Jo-Ken-Po totalmente em Python onde você pode escolher quantas rodadas deseja jogar e mostra o numero de "vitorias", de "derrotas" e o campeão de todas as rodadas.

Projeto Votação

Esse programa totalmente em Python simula uma eleição, ele mostra quantos votos cada candidato teve e se houve um capeão ou se houve empate entre os candidatos.

Projeto Jogo interativo

Um jogo interativo totalmente em Python, se chama "Super Batata" onde você precisa derrotar todos seus inimigos para ganhar o jogo. Conforme as suas respostas você pode treinar seu personagem e batalhar contra seus inimigos para avançar no jogo.
